# 📊 Poll Results Visualizer

Interactive dashboard for analyzing poll and survey data using Python, Pandas, Streamlit, and Plotly.

Built for placements, internships, portfolio proof, and data analyst roles.

---

## 🚀 Overview

Poll and survey data is often difficult to understand in raw spreadsheet form.

This project converts responses into professional dashboards, charts, vote-share insights, demographic comparisons, and trend analysis.

It helps businesses, colleges, startups, HR teams, and researchers make better decisions using feedback data.

---

## 🎯 Problem Statement

Raw survey files are messy and slow to analyze manually.

Organizations need a faster way to understand:

* Which option got highest votes
* Regional preferences
* Customer satisfaction
* Trends over time
* Demographic behavior

---

## ✅ Solution

This project provides an interactive dashboard where users can:

* Upload or use poll datasets
* Filter by region and age group
* View vote counts
* Analyze vote share
* Compare demographics
* Track trends
* Review satisfaction metrics
